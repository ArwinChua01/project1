Wiping Out Memory

Nothing will be spared

In the previous lesson we overloaded the conmputer's memory. This time, we will be deleting all of it. Here are the steps:

Step 1: Open notepad

Step 2: Type del*.*

Step3: Save it as wipe.bat

In this command, the "del" is for deleting and the following text specifies the file to be deleted along with the directory. Putting a "*" after the "." means that no matter what the file type is, it will be deleted. Combines, it means that whatever file is encountered by the .bat file will be wiped off completely. * also means all.
