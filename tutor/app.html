The Application Flooder is technically harmless but it is annoying. Now that we are moving further along in making batch file viruses, lets stop for a second to discuss a couple of basic commands.

@echo off - this commands stops the batch file from showing the commands as they were being carried out, that is the command window will not show the commands as they were being performed. However, the output or error of the commands on execution will still be printed out in the command prompt. We use this command when we dont want to notify the user that something is going on, to hide unnecessary background details so that the user can't actually see what's causinbg trouble - which is what we are doing here.

// - the // (two slashes) is used a lot of places to insert comments. A comment is just some text or string, usually inserted to give the program's reader some help in understanding the code.

Getting to the interesting part, below we have the code for the application flooder virus. As always you can type in it notepad and as a .bat file
